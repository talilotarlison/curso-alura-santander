A construção e compartilhamento de sistemas operacionais de código aberto baseados no Linux Kernel para diferentes plataformas tem promovido a inovação de maneira contínua e colaborativa. A flexibilidade do Linux Kernel permite a adaptação a uma ampla variedade de dispositivos, desde servidores com grande capacidade de processamento até sistemas embarcados mais simples. Por isso, é fundamental conhecermos bem como lidar com as diferentes distribuições Linux em ambientes de produção.

Dessa forma, criamos uma lista de atividades (não obrigatórias) para que você possa praticar ainda mais sobre os tópicos que exploramos ao longo dessa aula. Para realização dessas atividades, utilizaremos a máquina virtual com Ubuntu Server configurada na aula.

Atividades

Liste todos os arquivos e diretórios existentes dentro da home de sua máquina virtual.
Verifique a existência de arquivos ocultos em sua home.
Utilize um comando para exibir o path (caminho) do diretório em que você se encontra.
Percorra outros diretórios de sua máquina virtual.
Utilize um comando para exibir o conteúdo do diretório percorrido de forma mais detalhada.
Caso necessite de ajuda, opções de solução das atividades propostas estão disponíveis na seção “Opinião da pessoa instrutora”.

Opções
Deixamos aqui a solução para as atividades propostas para você comparar com suas respostas.

Atividade 1

Use o comando ls para obter a lista dos arquivos e diretórios existentes dentro de um diretório.

ls

Atividade 2

Utilize o comando ls com a opção -a para exibir nos resultados da listagem os arquivos e pastas ocultas existentes dentro do diretório.

ls -a

Atividade 3

O comando pwd retorna o caminho completo do diretório em que você se encontra.

pwd

Atividade 4

Para percorrer diferentes diretórios dentro do Linux, podemos usar o seguinte comando:

cd /caminho/do/diretorio/desejado

Não se esqueça de substituir o /caminho/do/diretorio/desejado pelo caminho do diretório para o qual você deseja navegar.

Atividade 5

Use o comando ls com a opção -l para obter uma listagem mais detalhada, incluindo permissões, proprietário, tamanho e data de modificação dos arquivos.

ls -l
