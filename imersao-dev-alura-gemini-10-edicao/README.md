
# 🌐 Base de Conhecimento — Imersão Dev Alura + Gemini

Este projeto foi desenvolvido durante a **Imersão Dev Alura + Gemini**, com o objetivo de criar uma **Base de Conhecimento** onde o usuário pode pesquisar linguagens de programação e visualizar informações organizadas em cards.
A aplicação foi construída utilizando **HTML, CSS e JavaScript puro**, focando em responsividade, experiência do usuário e boas práticas de código.

---

## 🚀 Funcionalidades

* 🔎 **Campo de busca inteligente**
  Permite pesquisar linguagens de programação ou tecnologias dentro da base.

* 🧠 **Exibição dinâmica dos resultados**
  Os dados são carregados e apresentados automaticamente na tela.

* 🎨 **Design moderno e responsivo**
  Inspirado no visual escuro minimalista, com animações leves e foco em acessibilidade.

* 📱 **Compatível com mobile**
  O layout se adapta bem a telas pequenas, removendo elementos que atrapalham a visualização.

---

## 🛠️ Tecnologias utilizadas

* **HTML5** — estrutura semântica limpa
* **CSS3** — estilização com variáveis, responsividade e animações
* **JavaScript** — lógica de busca e renderização dinâmica
* **Google Fonts** — fonte Quicksand para uma identidade visual agradável
